# 백엔드 함수 명세

프론트(`frontend/index_v2.html`)에서 `google.script.run`으로 호출하는 백엔드 함수 목록입니다.

---

## 1. getDashboardData(period)

### 목적
홈 대시보드 전체 데이터 반환

### 호출 방식
```javascript
google.script.run
  .withSuccessHandler(d => { ... })
  .getDashboardData(period); // 'today' | 'week'
```

### 입력
| 파라미터 | 타입 | 값 |
|---|---|---|
| `period` | string | `'today'` 또는 `'week'` |

### 반환 구조
```javascript
{
  filter:      'today' | 'week',
  today:       'YYYY-MM-DD',
  weekRange:   { start: 'YYYY-MM-DD', end: 'YYYY-MM-DD' },

  storeStatus: { text: string, updatedAt: string, author: string },

  salesGoal: {
    daily:   { goal, actual, rate, status },
    weekly:  { goal, actual, rate, status },
    monthly: { goal, actual, rate, status },
  },

  productFlow: {
    today: { topByQty: [], topBySales: [] },
    week:  { topByQty: [], topBySales: [] },
  },

  issues: {
    pending:      [],   // 미처리 이슈 목록
    pendingTotal: number,
  },

  noReasons: {
    today: { top3: [], total: number },
    week:  { top3: [], total: number },
  },

  feedback: {
    week: { satisfaction, recommendation, revisit, count },
  },

  safeSize: {
    today: { participants, conversions, rate },
  },

  snsEvent: {
    today: { participants, conversions, rate },
  },

  todayNotes:   [],   // 오늘의 특이사항
  todayActions: [],   // 오늘의 액션 항목
}
```

## 2. getTabData(tabType, filters)

### 목적
각 기능 탭의 KPI, 분석 지표, 필터 적용 리스트 반환

### 호출 방식
```javascript
google.script.run
  .withSuccessHandler(data => { ... })
  .getTabData(tabType, filters);
```

### 입력
| 파라미터 | 타입 | 설명 |
|---|---|---|
| `tabType` | string | `'issue'` \| `'noreason'` \| `'note'` \| `'safesize'` \| `'sns'` |
| `filters` | object | 아래 참고 |

**filters 구조 (tabType별)**

| tabType | 사용 가능한 filters |
|---|---|
| `issue` | `{ period, status }` |
| `noreason` | `{ period, category }` |
| `note` | `{ period, level, followUpOnly }` |
| `safesize` | `{ period, purchased }` |
| `sns` | `{ period, purchased }` |

- `period`: `'today'` \| `'week'` \| `'all'`
- `status`: `'전체'` \| `'미확인'` \| `'확인중'` \| `'처리완료'`
- `category`: `'전체'` \| `'신발'` \| `'의류'` \| `'용품'` \| `'기타'`
- `level`: `'전체'` \| `'높음'` \| `'보통'` \| `'낮음'`
- `followUpOnly`: `true` \| `false`
- `purchased`: `'전체'` \| `'구매O'` \| `'구매X'`

### 반환 구조 (공통)
```javascript
{
  kpi:       {},    // 탭별 KPI 3개
  breakdown: [],    // 유형별/사유별 분석
  list:      [],    // 필터 적용 데이터 리스트 (최신순)
  total:     number,
}
```

## 3. doPost_proxy(payload)

### 목적
프론트 폼 저장 요청 처리 (저장 타입별 라우팅)

### 호출 방식
```javascript
google.script.run
  .withSuccessHandler(result => { ... })
  .doPost_proxy(payload);
```

### 입력 — type별 payload 구조

#### type: 'issue' — 제품이슈 등록
```javascript
{
  type: 'issue',
  date: 'YYYY-MM-DD',        // 자동 (오늘)
  author: '승호 강',
  issueType: 'size_shortage', // erp_mismatch | size_shortage | product_damage
  brand: '나이키',
  product: '페가수스 42',
  color: '',
  size: '275',
  erpStock: '3',
  realStock: '0',
  content: '재고 없음',
}
```

#### type: 'issue_status' — 제품이슈 상태 변경
```javascript
{
  type: 'issue_status',
  rowIndex: 5,              // Sheets 행 번호
  status: '처리완료',       // 미확인 | 확인중 | 처리완료
  resolver: '승호 강',
  managerMemo: '본사 확인 완료',
}
```

#### type: 'noreason' — 미구매 사유 등록
```javascript
{
  type: 'noreason',
  author: '임창성',
  category: '신발',
  reason: '사이즈/재고 부족',
  brand: '온러닝',
  product: '클라우드몬스터 3',
  size: '270',
  memo: '',
}
```

#### type: 'note' — 특이사항 등록
```javascript
{
  type: 'note',
  author: '승호 강',
  content: '러닝 크루 단체 방문 예정',
  level: '높음',            // 높음 | 보통 | 낮음
  followUp: 'Y',            // Y | N
}
```

#### type: 'note_detail' — 특이사항 점장 수정
```javascript
{
  type: 'note_detail',
  rowIndex: 3,
  followUp: 'Y',
  managerMemo: '확인 완료',
}
```

#### type: 'safesize' — 세이프사이즈 등록
```javascript
{
  type: 'safesize',
  author: '승호 강',
  proceeded: 'Y',           // Y | N
  purchased: 'Y',           // Y | N
  products: [
    { name: '클라우드몬스터 3', discount: '10', price: '180000' },
    { name: '온 양말', discount: '', price: '35000' },
  ],
  runningPurpose: '대회 준비',
  memo: '발볼 설명 후 만족',
}
```

#### type: 'sns' — SNS 인증 등록
```javascript
{
  type: 'sns',
  author: '임창성',
  participated: 'Y',        // Y | N
  purchased: 'Y',           // Y | N
  products: [
    { name: '마인드 001', discount: '', price: '108182' },
  ],
  channel: '인스타그램',    // 인스타그램 | 블로그 | 기타
  benefitGiven: 'Y',        // Y | N
  memo: '',
}
```

#### type: 'setting' — 점장 설정 저장
```javascript
{
  type: 'setting',
  storeStatus: '오늘은 마감까지 신발 연계 제안을 강화합니다.',
  action1: '신발 구매 고객에게 양말 연계 제안',
  action2: 'ERP 재고 확인',
  action3: '',
  action4: '',
  action5: '',
  commentSales: '',
  commentProduct: '',
  commentNoReason: '',
}
```

### 반환
```javascript
{ success: true }
// 또는
{ success: false, error: '오류 메시지' }
```
