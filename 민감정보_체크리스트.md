# GitHub 업로드 전 민감정보 체크리스트

아래 항목이 코드에 포함되어 있지 않은지 확인 후 업로드해주세요.

---

## 절대 포함하면 안 되는 정보

- [ ] Google Apps Script 배포 URL (`script.google.com/macros/...`)
- [ ] Google Sheet ID (URL의 `/d/` 뒤 긴 문자열)
- [ ] Google 계정 이메일 주소
- [ ] API Key 또는 토큰
- [ ] 고객 개인정보 (이름, 연락처 등)
- [ ] 실제 매출 원본 데이터
- [ ] Google Form 실제 응답 원본
- [ ] ERP 실제 데이터 원본

---

## 코드 내 확인 방법

`index_v2.html`과 `dashboard_final.gs`에서 아래 패턴을 검색하세요.

```
script.google.com/macros   ← 배포 URL
/spreadsheets/d/           ← Sheet ID
@gmail.com                 ← 이메일
AIza                       ← Google API Key 접두사
```

---

## 안전한 처리 방식

하드코딩 대신 아래처럼 처리하세요.

```javascript
// ❌ 하지 말 것
const SHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms';

// ✅ 이렇게 처리
const SHEET_ID = 'YOUR_GOOGLE_SHEET_ID'; // 실제 값은 Apps Script 내부에서만 관리
```

실제 운영 값은 Apps Script 프로젝트 안의 스크립트 속성(Script Properties)이나  
별도 비공개 메모로 관리하세요.
