# 데이터 구조

Google Sheets 탭별 컬럼 구조입니다.  
모든 탭의 서식은 **일반 텍스트**로 설정합니다.

---

## 공통 규칙

- 날짜 형식: `YYYY-MM-DD` (예: `2026-05-22`)
- Y/N 값: 문자열 `'Y'` 또는 `'N'`
- 금액: 부가세 제외 숫자 (예: `152955`)
- 빈 값: 빈 문자열 `''`

---

## 탭 목록

| 탭명 | 역할 | 입력 주체 |
|---|---|---|
| `판매데이터_RAW` | ERP 엑셀 붙여넣기 원본 | 점장 |
| `판매데이터_CLEAN` | 자동 변환 정제 데이터 | Apps Script 자동 |
| `목표관리` | 일/주/월 목표 설정 | 점장 |
| `제품이슈` | 제품 이슈 등록 및 처리 | 직원/점장 |
| `미구매사유` | 미구매 고객 사유 기록 | 직원 |

## 판매데이터_CLEAN

ERP 원본(`판매데이터_RAW`)을 `processRawData()` 실행 시 자동 변환.

| 컬럼 | 타입 | 설명 |
|---|---|---|
| 날짜 | DATE | YYYY-MM-DD |
| 영수증번호 | TEXT | 거래 식별자 |
| 직원명 | TEXT | |
| 브랜드 | TEXT | |
| 카테고리 | TEXT | 신발/의류/용품/기타 |
| 실루엣 | TEXT | |
| 성별 | TEXT | |
| 상품명 | TEXT | |
| 품번 | TEXT | 상품 고유 식별자 |
| 사이즈 | TEXT | |
| 수량 | NUMBER | 음수 = 반품 |
| 매출액 | NUMBER | 부가세 제외 |
| 창고 | TEXT | `매장 (신사동)` 고정 |
| 거래유형 | TEXT | `정상판매` 또는 `반품` |

**필터 조건:** `수주구분 = 매장` + `창고 = 매장 (신사동)` 만 추출  
**반품 판별:** `수량 < 0` → `거래유형 = 반품` 자동 태깅

## 고객피드백

Google Form 응답이 자동으로 쌓이는 탭.

| Form 컬럼명 | 참조명 | 타입 |
|---|---|---|
| 타임스탬프 | 제출시간 | DATETIME |
| 오늘 구매하신 제품에 만족하시나요? | 제품만족도 | NUMBER (1~5) |
| 1. 오늘 방문에서 가장 도움이 되었던 건 무엇이었나요? | 도움된부분 | TEXT |
| 2. 추천받은 제품이 내 러닝 목적과 잘 맞는다고 느끼셨나요? | 추천적합도 | TEXT → 점수 변환 |
| 3. 오늘 경험 후 다시 방문하고 싶은 마음이 드셨나요? | 재방문의향 | TEXT → 점수 변환 |
| 4. 레이스먼트에서 더 기대하는 서비스가 있나요? | 기대서비스 | TEXT |
| 5. 아쉬웠던 점이나 기억에 남는 점이 있다면 자유롭게 남겨주세요 | 자유의견 | TEXT |

**타임스탬프 형식:** `2026. 5. 12 오후 12:03:42` → `_parseKoreanTimestamp()` 처리
